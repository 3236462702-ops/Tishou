# 全局强制约束

## 1. 运行载体
- **仅适配安卓真机**，不做 Windows、模拟器、云手机适配。

## 2. 开发语言
- **纯 Python 开发**，不混用 C/C++、NDK、Java、Kotlin 等语言。

## 3. 网络 & 接口
- **仅使用免费、免注册、无授权公共 API**。
- 禁用付费、私有、需登录接口。

## 4. 素材规则
- 仅使用 Python 开源库内置免费素材。
- 支持库内素材定时/手动更新。
- **不下载第三方独立资源包**。

## 5. 镜像规则（pip）
- 优先使用阿里云镜像：`https://mirrors.aliyun.com/pypi/simple/`
- 10秒超时，重试1次后自动切换腾讯云：`https://mirrors.cloud.tencent.com/pypi/simple/`
- **屏蔽境外源**（pypi.org / pypi.python.org）。
- 全程输出下载/切换日志，日志文件 `logs/pip_setup.log`。
- 实现文件：
  - `pip_setup.py` — 镜像测速、选择、全局配置、依赖安装、切换回退
  - `pip.ini` — 项目级 pip 配置文件（也可复制到 `%APPDATA%\pip\pip.ini` 全局生效）
- 安装依赖标准命令：`python pip_setup.py`
- 支持 `--auto` 自动模式、`--mirror aliyun|tencent` 指定镜像

## 6. 异常要求
- **全代码加异常捕获**，杜绝闪退、卡死。
- 配置文件损坏自动恢复默认。

## 7. 日志规范
- 分级日志：运行日志、错误日志、OCR日志、素材更新日志。
- **正式包关闭冗余调试日志**。

## 8. OCR 统一规范
- 使用 **easyocr + Pillow**，纯本地识别。
- 启动仅加载一次 `ch_sim` 简体模型。
- 关闭自动更新、禁用 GPU。
- 图像预处理流程：截取有效区域 → 灰度 → 二值化 → 降噪 → 放大1.3倍。
- 单次识别 **8秒超时**。
- 识别后释放图片资源。

## 9. 抢单判定延迟
- 默认 **5秒**，支持自定义。

## 10. UI 风格
- **HyperOS + iOS 融合风**。
- 主卡片 18~22dp 圆角、按钮胶囊圆角、悬浮窗 16dp 圆角。
- 背景色 `#F2F2F7`。
- 深浅主题跟随系统，可开关动画。
